# Offline MVP Scope

## Goal

Build a small offline match that proves Puck Clash is fun as hockey before adding online play or heavy systems.

## Included

- Local player versus CPU.
- Single rink scene.
- Player movement.
- Puck possession or controllable puck interaction.
- Shooting.
- Defending.
- Goals and score reset.
- Basic match end condition.
- Minimal HUD for score and match state.
- A small number of skills only after the hockey loop works.

## Not Included

- Online multiplayer.
- Vapor server.
- Firebase or any backend service.
- Authentication.
- Database.
- Ranking or matchmaking.
- Ads, analytics, monetization, or store flows.
- Cosmetic inventories.
- Complex league, season, or progression systems.

## Acceptance Direction

The MVP should answer these questions:

- Can the player understand who has the puck?
- Can the player move, shoot, defend, and score?
- Does the CPU create enough pressure to make the match useful for testing?
- Do skills add tactical choices without hiding the hockey fundamentals?
